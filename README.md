FloaterBall
===============

HTML5 Canvas game.  Navigate floating ball and collect squares.  Arrows to navigate, shift key to stop ball.  Use the walls to your advantage.  This is a rough start.....

#Required
1.  Timer

2.  Levels

3.  Obsticles

#Required
A browser.

#Install
1.  Download code.

2.  Open index.html in browser.
